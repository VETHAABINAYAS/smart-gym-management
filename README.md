# 🏋️ Fitness Club Management System

A simple web-based application to manage gym memberships. This project allows users to register for a fitness club by entering their personal details and selecting membership plans.

---

## 🚀 Features

* User Registration Form
* Stores data in MySQL database
* Clean and responsive UI
* Success message display after registration
* Header and Footer layout

---

## 🛠️ Technologies Used

* HTML
* CSS
* PHP
* MySQL
* WAMP Server

---

## 📂 Project Structure

```
fitness/
│── membership.html
│── register.php
│── db.php
│── style.css
```

---

## ⚙️ Setup Instructions

1. Install WAMP/XAMPP
2. Move project folder to:

   ```
   C:\wamp\www\
   ```
3. Start Apache & MySQL
4. Create database:

   ```
   fitness
   ```
5. Create table:

   ```
   membership
   ```
6. Open browser:

   ```
   http://localhost/fitness/membership.html
   ```
